# KonexZero Cloud

This repository contains the open-source core of KonexZero — a fully functional WebRTC signaling platform you can self-host.

**Want more?** [KonexZero Cloud](https://konexzero.com) adds:

- **Developer Dashboard** — Manage applications, API keys, and webhooks through a web UI
- **Real-time Analytics** — Call volume, room activity, participant metrics
- **Usage Monitoring** — Track consumption across your applications
- **Team Management** — Multi-user organizations with role-based access
- **Global Infrastructure** — Deployed on Cloudflare's edge network across 300+ locations
- **Automatic Scaling** — No capacity planning needed
- **99.9% SLA** — Enterprise-grade reliability
- **Priority Support** — Direct access to the engineering team
